## Pinterest React (Next.js) Clone

### How to run localside

```bash
# Clone this repository
$ git clone https://github.com/korayguler/pinterest-clone

# Go into the repository
$ cd pinterest-clone


# Install dependencies
$ npm install

# Start the backend server
$ npm run dev

```

- next
